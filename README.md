# Shopify Spin Wheel App

A complete, modern, and interactive spin wheel application for Shopify merchants. This app helps engage customers, grow email lists, and boost conversions through gamified discount codes.

## üéØ Features

### User-Facing Spin Wheel
- **Interactive Wheel**: Smooth animations with 8 prize segments
- **Vibrant Design**: Modern color palette (teal, purple, orange, yellow)
- **Email Validation**: Collects user emails before revealing prizes
- **Prize Types**: 
  - Percentage discounts (10%, 15%, 20%, 25% OFF)
  - Free Shipping
  - Fixed amount discounts ($5 OFF)
  - Try Again segments
- **Discount Codes**: Auto-generated codes displayed in modal
- **Responsive**: Works seamlessly on desktop and mobile

### Admin Dashboard
- **Campaign Management**:
  - Create and manage multiple campaigns
  - Set start/end dates
  - Configure spin limits
  - Enable/disable campaigns
  - Track usage with progress bars
  
- **Analytics Dashboard**:
  - Total spins counter
  - Discount codes redeemed
  - Conversion rate tracking
  - Revenue generated
  - Line charts for daily performance
  - Pie charts for prize distribution
  - Bar charts for campaign comparisons
  
- **Pricing Plans**:
  - Free Plan: 100 spins/month
  - Pro Plan: 1,000 spins/month ($29/month)
  - Enterprise Plan: Unlimited spins ($99/month)
  - Feature comparison cards
  - FAQ section

### Landing Page
- Hero section with call-to-action
- Feature highlights with icons
- Statistics showcase
- Benefits overview
